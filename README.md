# H.D.D System
这是一个还原游戏《绝区零》中的H.D.D系统的 GUI 程序，使用Qt C++实现。

## 使用方法
下载Release中对应平台的文件后运行即可。

或自行安装Qt5.15.x，编译运行。

目前仅支持 Windows 和 Linux 平台。
